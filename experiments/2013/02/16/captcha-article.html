Un-user friendly 'CAPTCHA'.

Publishing date: 16 02 2013

experiments

  • future
  • talking

    • As it's said on captcha.net:

    "A CAPTCHA is a program that protevys websites against bots by generating grading tests that humans can pass but current computer programs cannot. For example, humans can read destorted text as the one shown below, but current computer programs can't."

    The project started in 2000. It started with the purpose of mark the difference between a human and a bot.

    Most of the times, the process becomes difficult and blocks the user, instead of making it easier.

    After all, the system is in place to prove that the user is human, not to test the nation's patience, eye-sight or ability at word jumbles.

    This is a relevant problem for who's interested in UX (User Experience) but there is a solution: sipler images, and verification systems still valuable.

    Such as MotionCAPTCHA, visualCaptcha or The Puzzle CAPTCHA, and amanuman that still ensure a good level of security using games or gestures instead.
